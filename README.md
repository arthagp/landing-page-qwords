## REDESIGN LANDING PAGE QWORDS (Rakamin x Qwords : Project Based Internship)

**Tech Stack:**
- Laravel
- Tailwind CSS

**How To Use:**

1. Install all packages with:

    ```
    composer install
    npm install
    ```

2. Run the project:

    ```
    php artisan serve
    ```

3. I used Tailwind, so just type the following command to run:

    ```
    npm run dev
    ```
